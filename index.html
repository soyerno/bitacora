Herramientas · mi stack en Claude Code

Skills, MCPs y plugins que uso día a día en Claude Code para sostener el trabajo de TL en MODO. Agrupados por uso real, no por orden alfabético. Cada uno se invoca por slash o auto-trigger según corresponda. Markers: MCP servidor MCP · plugin plugin Claude Code · github skill comunitario externo.

MODO · skills y MCPs propios

modo-deck
Orquestador de decks HTML branded MODO. Voz rioplatense, Pyramid Principle, comms discipline. Punto de entrada para todo deck en este repo.
modo-design-system
SDK Next/Astro para generar páginas one-shot con tokens canónicos MODO (Quicksand + Red Hat Display, #008859). Auto-invoca en cualquier repo del workspace.
modo-tech-architect
PRD → specs OpenSpec + plan de implementación + sprints + tareas Jira. Consulta APIs Swagger, Confluence y repos MODO para detectar reuso.
modo-code-review-checklist
Checklist de review tailored a patrones modo-landing. Auto-invoca en PRs para chequear quality, security y consistencia arquitectónica.
modo-promosMCP
Consulta promos vigentes, bancos y comercios adheridos en cualquier ambiente (develop/qa/preprod/prod). Tools bajo mcp__modo-promos__*.
consultar-promos-modo
Capa NL sobre modo-promos. Mapea preguntas humanas ("¿qué descuentos hay hoy en supermercados?") a las tools correctas del MCP.
simular-compra-agentica
Demo end-to-end del flow agentic MODO Checkout (POC del hackathon): discovery, Intent Mandate, Cart Mandate, SPT, deeplink, biometric.
agente-cx-whatsapp
Operar el agente CX/Promos por WhatsApp (OpenWA + Anthropic + Confluence + modo-promos + anti-bot). Trigger por frases tipo "levantar el agente cx".

AI · workflow + memoria + agentes

rufloMCP
Memoria persistente (sql.js + HNSW + ONNX), swarm coordinator, autopilot y federation cross-installation. Backend único de memoria user-level.
claude-api
Build/debug apps con Claude API + Anthropic SDK. Maneja prompt caching, tool use, batch, files, citations y migración entre versiones de modelo.
mcp-builder
Diseñar, construir y testear MCP servers custom con tools, resources y prompts para extender capabilities de agentes.
skill-creator
Crear skills nuevos (o actualizar existentes) que extienden Claude con conocimiento especializado, workflows o integraciones.
sdd-* · explore / propose / design / tasks / apply / verify / archive
Spec-Driven Development en MODO. Obligatorio para cambios en /comercios y features no-triviales: spec antes de código, verify después.
brainstorming
Explorar intent + requirements + design ANTES de implementar. Bloquea cualquier feature/componente/behavior nuevo hasta tener intent claro.
doc-coauthoring
Workflow estructurado para escribir docs, proposals, technical specs y decision docs. Transfer de contexto + iteración + verificación con readers.
cavemanplugin
Comms compressed ~75% token savings para agent-to-agent. Sub-agents de guardia usan caveman entre ellos; user-facing output sigue siendo prose.

Frontend · diseño y UX

impeccablegithub
Skill de Phil Bakaus para frontend production-grade. Anti-patterns checks: sin gradientes, sin nested cards, color restrained, sin em dashes.
taste-skillgithub
Anti-slop frontend rules de Leonxlnx. Evita over-design y mantiene interfaces limpias y funcionales.
ui-ux-pro-maxgithub
161 paletas, 57 font pairings, 50+ estilos de UI (NextLevelBuilder). Referencia para composición visual y tipografía.
frontend-design
Diseño visual + interacción para web. Cobertura amplia: layouts, hierarchy, motion, micro-interactions.
web-design-guidelines
Review de UI contra Web Interface Guidelines. Trigger en frases tipo "review my UI", "check accessibility", "audit design".
react-doctor
Catch React issues early después de cambios. Se invoca al revisar código, terminar features o fixear bugs en proyectos React.
