Bitácora

Digest diario de aprendizajes. Cada día es una página navegable con un TLDR arriba y una historia por feature o lección — voz rioplatense, claridad pedagógica estilo Karpathy. La idea es la del LLM Wiki Pattern: escribir lo aprendido para no tener que re-aprenderlo. Datos crudos en bitacora.json para que un agente consuma el catálogo sin parsear el HTML.

📜 Timeline de PRs — registro append-only de cada pull request abierto: qué hace y qué dolor resuelve. Refresh automático 09:00 + 16:30 ART.
