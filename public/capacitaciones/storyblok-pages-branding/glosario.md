# 📖 Glosario · técnico → humano

> Dominar estas palabras te deja hablar de igual a igual con desarrollo y diseño. No para entender de afuera: para pedir, decidir y construir con vocabulario propio.

## Contenido y páginas

- **Página / “story”** — Una pantalla completa del sitio (la home, una landing de producto, una sección). En la herramienta de contenido se la llama *story*.
- **Bloque / “blok”** — Una sección reutilizable de la página: encabezado, grilla, banner, listado. Se construye una vez y se reusa en muchas páginas.
- **Dirección / URL / “slug”** — Lo que va después de `modo.com.ar/`. Identifica la página y es lo que compartís.
- **CMS / Storyblok** — La herramienta donde se carga y edita el contenido sin programar.
- **Borrador / “draft”** — Versión visible solo para el equipo, todavía no publicada.
- **Publicar** — Dejar el contenido visible para el público.

## Marca y diseño

- **Sistema de diseño / design system** — El conjunto único de colores, tipografías y componentes de MODO. Todas las páginas toman su estilo de ahí.
- **Token** — Cada definición de marca con nombre (por ejemplo, “verde primario”). Se usa el nombre, no el color exacto a mano, para mantener coherencia.
- **Accesibilidad / a11y** — Que la página sea usable por todos, incluida la gente con discapacidad (contraste, texto alternativo, navegación por teclado).
- **Texto alternativo / “alt”** — La descripción corta de una imagen. La leen los lectores de pantalla y Google. Nunca va vacía.

## Calidad y publicación

- **Control de calidad / “gate”** — Cada una de las cuatro revisiones que una página debe pasar antes de publicarse. Si una está en rojo, no se publica.
- **Revisión entre pares / code review** — Otra persona del equipo revisa el trabajo antes de aprobarlo.
- **Performance / velocidad** — Cuánto tarda la página en aparecer y si se mantiene estable mientras carga.
- **Seguridad / CSP** — Reglas que aseguran que la página solo cargue contenido de fuentes confiables.
- **SEO / GEO** — Que la página se pueda encontrar en Google (SEO) y en los buscadores con inteligencia artificial (GEO).
- **Ambiente de prueba / alpha** — Una copia interna del sitio, idéntica a la real, para probar antes de exponer al público.
- **Producción** — El sitio real, modo.com.ar, el que ve la gente.

## Roles (a grandes rasgos)

- **Branding / diseño** — Cuida que todo sea de marca y se vea bien.
- **Contenido / marketing** — Escribe los textos, arma las campañas, elige imágenes.
- **Desarrollo** — Construye los bloques nuevos, publica a producción, cuida velocidad y seguridad.
